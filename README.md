# Personal Portfolio Website - Nandhini Pothunuri

A modern, responsive, and professional Personal Portfolio Website developed for **Nandhini Pothunuri**, an Engineering Student and Aspiring Full Stack Developer.

This website is styled using a premium **dark glassmorphism theme** with elegant **gold accents (`#f6c453`)**, custom illustrations, and smooth micro-animations.

---

## 🚀 Key Features

- **Responsive Design**: Optimized for seamless viewing on desktop, tablet, and mobile devices.
- **Glassmorphic Sticky Sidebar**: Displays personal profile avatar, professional summary, contact details, and social links. On mobile, it collapses into a slide-down menu drawer.
- **Floating Navigation Navbar**: Tab-switching interface for fluid browsing between tabs:
  - **About**: Profile overview and services ("What I'm Doing" cards).
  - **Resume**: Vertical double-column timeline highlighting education and virtual internship experience, coupled with viewport-animated skill progress bars.
  - **Projects**: Clean card grid featuring screenshots, technology badges, and links.
  - **Certifications**: Gallery of certificates featuring a custom glassmorphism lightbox pop-up viewer modal.
  - **Contact**: Interactive panel containing contact cards, an inverted dark Google Map, and a validated contact form showing loading indicators and success toasts.

---

## 🛠️ Technologies Used

- **Frontend Core**: HTML5, CSS3, JavaScript (Vanilla ES6)
- **Typography & Icons**: Poppins (Google Fonts), FontAwesome Icons
- **Mapping**: Google Maps API Embed (custom-styled via CSS filter)
- **Styling Method**: CSS Grid, Flexbox, custom variables, `@keyframes` transitions, backdrop blur filters

## 💻 How to Run Locally

Since this is a static frontend website, no installation or compilation is needed:
1. Clone or download this repository.
2. Open the **`index.html`** file in any web browser (Chrome, Edge, Firefox, Safari) by double-clicking it, or drag and drop it into an open browser window.
